Interactive Bar Chart (D3 Visualization)

This visualization is part of a data visualization assignment where I created a bar chart using the D3 JavaScript library. The chart visualizes data from a CSV file and includes interactive tooltips that appear when the user hovers over a bar. This interaction allows viewers to explore the data more easily by seeing exact values without cluttering the chart with labels.

Working on this project helped me understand how data visualization libraries like D3 transform data into visual elements using scales, axes, and SVG graphics. I also implemented a tooltip interaction that dynamically updates its position and content depending on the user's cursor location.
