D3 Interactive Bar Chart

This page includes a bar chart created using the D3.js data visualization library. The chart loads data from a CSV file and generates an SVG-based visualization that allows users to explore the data visually.

I also implemented an interactive tooltip so that when a user hovers over a bar, the exact data value is displayed dynamically. This improves the readability of the chart and demonstrates basic interactive visualization techniques.
